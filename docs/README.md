# Hopback documentation

Guides for people who want to help run the network.

- [Running an agent](running-an-agent.md) — connect your MeshCore node to a
  regional Hopback instance so it can act as a diagnostics endpoint.

For a live view of every endpoint and agent on an instance, open its **Operator
status** page (linked in the site footer, at `/status`). It needs no sign-in.
